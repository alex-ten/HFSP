Glossary

Throughout the website, the following terms are used consistently:

  • Condition: level of informedness, either informed or uninformed (i+, i-)
  • Group: exploration sttrategy, either free or strategic (F, S)
  • Task: one of the four classification difficulty levels:
    • 1D: a single variable dimension that predicts class
    • I1D: two variable dimensions, but only one predicts class
    • 2D: two variable dimensions, a combination of which predicts class
    • R: two variable dimensions, none of which predict class
  • PC: percent correct
